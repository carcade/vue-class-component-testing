## Установка
Установить зависимости
```shell script
npm install
```

## Использование
Запуск встроенного веб-сервера
```shell script
npm run serve
```

Запуск development сборки
```shell script
npm run build
```

Запуск production сборки
```shell script
npm run prod
```

Запуск сборки с отслеживанием изменений
```shell script
npm run watch
```
